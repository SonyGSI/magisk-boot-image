# Magisk Boot Image

A collection of stock boot images pre-modified by Magisk.

## Create your own boot image

You will need three tools:

- [XperiFirm]
- [UnSIN]
- [Magisk Manager]

Create your boot image:

- download a firmware with [XperiFirm]
- unpack the compressed boot image with [UnSIN]
- copy the unmodified boot image to your device
- modify the boot image with the [Magisk Manager]
- copy the modified boot image to your PC
- flash the modified boot image in fastboot mode

[XperiFirm]: https://forum.xda-developers.com/crossdevice-dev/sony/pc-xperifirm-xperia-firmware-downloader-t2834142
[UnSIN]: https://forum.xda-developers.com/crossdevice-dev/sony/tool-unsin-sin-v3-unpacker-t3128106
[Magisk Manager]: https://magiskmanager.com/
